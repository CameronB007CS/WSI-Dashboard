Parser Engine — Algorithmic Complexity
The core parser uses a compiled regex with named capture groups, applied in a single linear pass. Each .sub file block is matched once — no backtracking, no re-scanning.
# Named-group regex — compiled once at class level
_FIELD_RE = re.compile(
r"""
Frequency\s*:\s*(?P<freq_val>\d+) |
Protocol\s*:\s*(?P<proto_val>[^\n]+) |
Bit\s*:\s*(?P<bit_val>\d+) |
Key\s*:\s*(?P<key_val>[0-9A-Fa-f\s]+)
""", re.VERBOSE | re.IGNORECASE)
