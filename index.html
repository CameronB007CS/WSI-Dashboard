Parser Engine — Regex Architecture
The core parser uses a compiled regex with named capture groups applied in a single linear pass. Each .sub file block is matched once — no backtracking, no re-scanning. Compiled once at class level.
# Named-group regex — compiled once at class level (O(1) amortised) _FIELD_RE = re.compile(r""" Frequency\s*:\s*(?P<freq_val>\d+) | Protocol\s*:\s*(?P<proto_val>[^\n]+) | Bit\s*:\s*(?P<bit_val>\d+) | Key\s*:\s*(?P<key_val>[0-9A-Fa-f\s]+) | Preset\s*:\s*(?P<preset_val>[^\n]+) """, re.VERBOSE | re.IGNORECASE)
